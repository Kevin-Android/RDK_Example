# Ui Extensions 组件

Ui Extensions的核心功能主要通过模块、工具处理器和注释处理器来实现。

这些工具处理器和注释处理器通常在加载相关功能模块时会被初始化并注册到UIExtensionsManager。

下面这个集合能够帮助您迅速找到相关功能代码。

# 模块
